# DevTool
DevTool

# Usage

1. Docker Start
```
docker-compose up -d --build
```

## React
* 起動 http://localhost:3000/
```
npm run start
```

* ビルド
```
npm run build
```

* テスト
```
npm run test
```

* エジェクト
```
npm run eject
```
